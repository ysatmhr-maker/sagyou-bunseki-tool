# 簡易作業動画分析ツール

動画を見ながら要素作業を打刻・集計できる、ブラウザで動く作業分析ツールです。  
インストール不要・完全ローカル処理・無料で使えます。

---

## 使い方

### ブラウザで今すぐ使う
→ [ここをクリック](https://ysatmhr-maker.github.io/sagyou-bunseki-tool/簡易作業動画分析アプリ.html)

### ダウンロードして使う（オフライン環境向け）
→ [HTMLファイルをダウンロード](https://github.com/ysatmhr-maker/sagyou-bunseki-tool/blob/main/簡易作業動画分析アプリ.html?raw=true)

ダウンロードしたHTMLファイルをダブルクリックするだけで起動します。

---

## 主な機能

- **動画再生** — ローカルの動画ファイルをドラッグ＆ドロップで読み込み。再生速度調整（×0.25〜×2）、コマ送り対応
- **打刻記録** — 再生中にEnterキーまたはボタンで現在時刻を記録。要素作業ごとの開始・終了・所要時間を自動計算
- **区間リピート** — 記録した区間を個別にリピート再生して確認可能
- **分析** — 円グラフ・統計サマリー・複数観測の比較テーブル
- **Excel出力** — 生データ・要素別集計・サマリーを1ファイルに出力
- **レイアウト切替** — 縦長動画向け・横長動画向けの2レイアウト

## 技術仕様

| 項目 | 内容 |
|---|---|
| 構成 | HTML / JavaScript 単一ファイル |
| 外部ライブラリ | [SheetJS](https://sheetjs.com/)（Excel出力）、[Chart.js](https://www.chartjs.org/)（グラフ）をCDN経由で読込 |
| 外部通信 | **一切なし**（動画・記録データはブラウザ外に送信されません） |
| データ保存 | **ブラウザ内メモリのみ**（ローカルストレージ不使用）。ページを閉じるとデータは消えます |
| 動作環境 | Chrome / Edge / Firefox など主要ブラウザ最新版 |

---

## セキュリティ・プライバシーについて

このツールは**完全にローカルで動作**します。

- 読み込んだ動画ファイルはブラウザ外に送信されません
- 記録・分析データは外部サーバーに保存されません
- ページを閉じるとすべてのデータはメモリから消去されます

工場内の作業動画など機密性の高い映像にも安心してご使用いただけます。  
ただし、ご利用の端末・ブラウザのセキュリティ管理はご自身でお願いします。

## オフライン環境での利用について

HTMLファイルをダウンロードしてご利用の場合、**初回起動時にインターネット接続が必要**です。  
これはグラフ描画（Chart.js）とExcel出力（SheetJS）のライブラリをCDNから読み込むためです。

> **完全オフライン環境で使いたい場合**  
> ライブラリをローカルに組み込んだ版が必要です。ご要望があればIssueやコメントでお知らせください。
